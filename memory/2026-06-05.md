## Fix: Envío de facturas enviaba PDF desactualizado

### Bug
Cuando se emitía/cambiaba estado de una factura, el PDF se había generado previamente en estado **borrador**. El código solo regeneraba el PDF si no existía o estaba corrupto (< 1KB), por lo que enviaba la versión antigua.

### Fix
Se modificaron **ambas funciones** en `api_invoices.py` para **siempre regenerar el PDF** antes de enviar:
- `send_invoice_whatsapp()` (envío manual)
- `_send_invoice_whatsapp_background()` (auto-notificación al pagar)

Antes: `if not os.path.exists(pdf_path) or os.path.getsize(pdf_path) < 1000: ... generate ... else: pdf_ok = True`
Ahora: `os.remove(pdf_path)` → siempre genera nuevo PDF
