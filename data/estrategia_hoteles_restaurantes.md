## 📊 Matriz de Decisión

```
Hoteles independientes (20-80 hab) → 🟢 Alta prioridad → Dueño local, decide rápido
Hoteles independientes (80-150 hab) → 🟡 Media prioridad → Más estructura, pero aún accesibles
Hoteles cadena nacional → 🟡 Media → Decisor regional, puede negociarse
Hoteles cadena internacional → 🔴 Evitar → Burocracia, exigencias formales

Restaurantes independientes → 🟢 Alta prioridad → Dueño operador
Catering empresarial → 🟢 Alta → Contratos grandes, decisión local
Cadenas restaurantes nacionales → 🟡 Media → Decisión centralizada
Franquicias → 🔴 Evitar → Procesos rígidos
```
